Neuro Racer

NeuroRacer was developed to be able to drive autonomously through a marked route.

The model car was controlled by a neural network that converted camera images and lidar sensor data directly into steering movements.

FLAS

Sorting colors in a 2d plane with the fast linear assignment sorting algorithm.

As you can see, FLAS can also be used to sort high dimensional features (like images features, produced by image encoders e.g. CLIP).

Video Pixelator

Original video by Frank Vincentz, CC BY-SA 3.0, via Wikimedia Commons

Originally developed to pixelate parts of videos, this tool can be used to create bounding boxes in videos and automatically track moving objects over time.

Uses the SIFT-Algorithm of opencv.

Object detection on medical data
